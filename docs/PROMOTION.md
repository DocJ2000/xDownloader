## Forum Post - English

```markdown
# xDownloader: bilingual local X/Twitter media downloader and browser

Hi everyone, I’m building xDownloader, an open-source Windows tool for downloading and browsing X/Twitter media locally.

The idea is simple: instead of only downloading files into folders, xDownloader also gives you a local browser UI with a timeline, media library, user list management, and X list sync.

Current features:

- English / Chinese bilingual UI
- Windows installer with selectable install folder
- Local media library and timeline view
- Multi-list sync from X list links
- Frontend settings page with Apply button
- Pause / continue / terminate download controls
- Optional shutdown after normal download completion
- No-loss upgrade flow for installer users

Privacy note: account credentials and downloaded media stay on your own machine. The app runs a local UI at `127.0.0.1`.

GitHub:
https://github.com/DocJ2000/xDownloader

Feedback, issues, and contributions are welcome.
```
