## First Issue Ideas / 初始 issue 建议

Creating a few public issues makes the project look more active and gives contributors an easy starting point.

创建几个公开 issue 可以让项目看起来更活跃，也方便贡献者上手。

Suggested issues:

建议 issue：

1. `Improve media playback controls / 改进媒体播放控制`
2. `Add safer Cookie import guide / 增加更安全的 Cookie 导入说明`
3. `Improve installer smoke tests / 改进安装包冒烟测试`
4. `Add more screenshots to README / 为 README 增加更多截图`
5. `Refine English and Chinese copy / 优化中英文文案`
